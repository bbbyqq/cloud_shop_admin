# cloud_shop_admin 云商城后台管理系统

# 运行方法：

## 1.在命令行输入 npm install 下载 node_moudle 文件夹
## 2.在命令行输入 npm run dev 启动项目

# 说明：

## 1.此项目为纯前端项目，并未调用任何后端接口，
## 2.使用UI组件为：ElementUI
## 3.数据使用的是前端mock.js模拟后台数据
## 4.项目仅适配PC端
